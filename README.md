# Route_Map
Aplicação para criação de rotas de forma interativa. Por meio dessa aplicação pode-se gerar as melhores rotas, podendo interagir com o mapa, criar e apagar camadas. Também permite a importação de arquivo shapefile, desde que zipado. Dessa forma, o usuário pode importar o seu polígono de área de estudo e criar rotas no limite desejado.
